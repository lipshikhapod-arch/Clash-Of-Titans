# Clash-Of-Titans
A modern MOBA gaming website built using HTML, CSS, and Bootstrap.

## Features
- Responsive design
- Modern gaming-inspired UI
- Glassmorphism cards
- Smooth animations and hover effects
- Hero section with game roles
- Interactive navigation and footer

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## Project Status
Completed and continuously improving with new features.
